# Bluesky Movie Poster Bot

Automatically posts organic, fan-style movie content with TMDB poster images to multiple Bluesky accounts. Posts are generated by DeepSeek via NVIDIA NIM.

## Features

- Picks a **different random movie** for each Bluesky account per run
- Sources from trending, top-rated, hidden gems, now playing, and genre picks
- AI-generated posts that sound like a real movie fan (no spam language, no emojis)
- Attaches the official **movie poster image** to each post
- `mycinebd.com` is always a **clickable hyperlink** in the post
- Runs **4× daily** via GitHub Actions schedule (or trigger manually)

## Setup

### 1. Fork / clone this repo to your GitHub account

### 2. Add GitHub Secrets

Go to **Settings → Secrets and variables → Actions → New repository secret**

| Secret name      | Value                          |
|------------------|-------------------------------|
| `TMDB_API_KEY`   | Your TMDB API key              |
| `NVIDIA_API_KEY` | Your NVIDIA NIM API key        |
| `BSKY_HANDLE_1`  | `yourhandle1.bsky.social`      |
| `BSKY_PASSWORD_1`| App password for account 1     |
| `BSKY_HANDLE_2`  | `yourhandle2.bsky.social`      |
| `BSKY_PASSWORD_2`| App password for account 2     |
| `BSKY_HANDLE_3`  | *(optional)* handle 3          |
| `BSKY_PASSWORD_3`| *(optional)* password 3        |
| `BSKY_HANDLE_4`  | *(optional)* handle 4          |
| `BSKY_PASSWORD_4`| *(optional)* password 4        |
| `BSKY_HANDLE_5`  | *(optional)* handle 5          |
| `BSKY_PASSWORD_5`| *(optional)* password 5        |

> **Bluesky app passwords**: Go to Settings → App Passwords on bsky.social and create one per account. Do NOT use your main password.

> **TMDB API key**: Free at https://www.themoviedb.org/settings/api

> **NVIDIA NIM API key**: Get it at https://integrate.api.nvidia.com

### 3. Enable Actions

Go to the **Actions** tab in your repo and enable workflows if prompted.

### 4. Test manually

Click **Actions → Bluesky Movie Poster Bot → Run workflow** to test immediately.

## Posting Schedule

Default: **4 times a day** at 6 AM, 12 PM, 6 PM, 10 PM UTC.

Edit `.github/workflows/post.yml` to change the `cron` lines.

## Post Style

- Focused on trending movies, hidden gems, plot twists, shocking endings, underrated series
- Natural American tone — sounds like a real movie fan
- 2–4 relevant hashtags
- Always ends with a clickable `mycinebd.com` link
- No emojis, no spam words, every post unique

```bash
export TMDB_API_KEY=your_key
export NVIDIA_API_KEY=your_key
export BSKY_HANDLE_1=you.bsky.social
export BSKY_PASSWORD_1=your-app-password

```
